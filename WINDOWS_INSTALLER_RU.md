# Windows-установщик GhostNet Cyber VPN

## Что теперь создаёт GitHub Actions

```text
GhostNet-Cyber-VPN-Setup.exe
version.json
```

Обычный переносимый ZIP больше не используется как основной Windows-релиз.

## Что делает установщик

- устанавливает приложение в профиль текущего пользователя;
- не требует прав администратора;
- создаёт ярлык в меню «Пуск»;
- предлагает создать ярлык на рабочем столе;
- регистрирует протокол `ghostnet://`;
- корректно возвращает пользователя из браузера после оплаты;
- добавляет стандартное удаление через «Установленные приложения» Windows;
- поддерживает установку новой версии поверх старой;
- запускает приложение после завершения установки.

## Куда загрузить на сайт

```text
/downloads/GhostNet-Cyber-VPN-Setup.exe
/downloads/version.json
```

Сначала загрузить установщик, затем `version.json`.

## Локальная сборка

Установить Inno Setup 6, затем запустить:

```text
build_windows.bat
```

Готовый файл:

```text
release/GhostNet-Cyber-VPN-Setup.exe
```

## Важно

Установщик пока не подписан цифровым сертификатом. Windows SmartScreen может
показать предупреждение «Неизвестный издатель». Для удаления предупреждения
позже потребуется сертификат подписи кода.
