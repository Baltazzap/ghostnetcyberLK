# Реферальная система в приложении

Добавлено:

- раздел «Реферальная система» в личном кабинете;
- личный реферальный код пользователя;
- копирование кода и ссылки;
- ввод реферального кода до первой покупки;
- статистика: приглашено, оплатили, начислено дней;
- в админ-панели компактный блок рефералов;
- промокоды остаются отдельной системой скидок.

Логика:

- реферальный код не даёт скидку покупателю;
- покупатель может использовать обычный промокод, например WELCOME;
- пригласивший получает +7 дней после первой успешной оплаты друга;
- пробный доступ, отменённые платежи и оплаты меньше 100 ₽ не засчитываются.
