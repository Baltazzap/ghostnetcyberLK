# GhostNet Cyber VPN — встроенный VPN Android

Добавлено:

- новая вкладка `VPN` в нижнем меню и боковом меню;
- автоматическая загрузка активных подписок из API;
- автоматический разбор VLESS/VMess/Trojan/SS ссылок из `vpn_key`;
- если ключей нет в `vpn_key`, приложение пробует загрузить подписку по `subscription_url`;
- выбор сервера внутри приложения;
- кнопка `Включить VPN` / `Отключить VPN`;
- запрос системного разрешения Android VPN;
- показ статуса, времени подключения и скорости;
- проверка пинга выбранного сервера;
- Windows-сборка остаётся кабинетом, встроенный VPN сейчас активен только на Android.

Важно:

- для Android используется пакет `flutter_v2ray`;
- пользователь больше не обязан копировать ключ в Hiddify/v2RayTun для Android;
- старый раздел ключей в кабинете оставлен как резервный вариант.
