# План нативного VPN core

Эта папка оставлена под следующую интеграцию Android VPN core.

Цель:

- заменить тестовый flutter_v2ray на sing-box / libbox;
- использовать Android VpnService;
- подключать рабочие VLESS-ссылки, которые уже работают в Happ / Hiddify / v2RayTun;
- не показывать статус «VPN включён», пока core реально не поднял туннель.

Текущая сборка — безопасный переходный вариант: ключи доступны, фейковое подключение отключено.
