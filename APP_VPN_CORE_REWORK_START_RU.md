# GhostNet Cyber VPN — переход на новый VPN core

В этой сборке старый встроенный модуль VPN отключён, потому что он запускал Android VPN-сервис и таймер, но не создавал реальное подключение к 3x-ui.

Что изменено:

- удалена зависимость `flutter_v2ray` из `pubspec.yaml`;
- убран запуск `startV2Ray`, `stopV2Ray`, `getServerDelay`;
- вкладка VPN больше не показывает фейковый статус подключения;
- серверы и подписка остаются доступны для копирования;
- пользователь может скопировать подписку или отдельный VLESS-сервер и добавить в Happ / Hiddify / v2RayTun;
- структура проекта подготовлена для следующего этапа: нативный Android VPN core на sing-box / libbox.

Следующий этап:

1. добавить Android native service;
2. подключить sing-box/libbox;
3. генерировать sing-box config из VLESS-ссылки;
4. включать VPN только после реального старта core;
5. показывать ошибку core, если подключение не поднялось.
