# Автоматическое обновление GhostNet Cyber VPN

## Что работает

- приложение проверяет обновление после запуска;
- проверка работает на Android и Windows;
- в личном кабинете есть кнопка «Проверить обновления»;
- обычное обновление можно отложить;
- обязательное обновление нельзя закрыть;
- Android открывает прямую ссылку на APK;
- Windows открывает прямую ссылку на ZIP;
- запрос `version.json` отправляется без кэша;
- при отсутствии интернета автоматическая проверка не мешает запуску.

## Куда загрузить файлы на сайт

В папку сайта:

```text
/downloads/
```

загрузить:

```text
GhostNet-Cyber-VPN.apk
GhostNet-Cyber-VPN-Windows.zip
version.json
```

Файл `version.json` должен открываться по адресу:

```text
https://ghostnetcyber.ru/downloads/version.json
```

Готовый файл лежит здесь:

```text
website_update_files/downloads/version.json
```

## Как выпускать следующую версию

1. Изменить версию в `pubspec.yaml`, например:

```yaml
version: 1.0.4+5
```

2. Собрать Android и Windows через GitHub Actions.
3. Скачать оба артефакта.
4. Загрузить новые APK и ZIP в `/downloads/` с теми же именами.
5. Загрузить новый `version.json` последним.

Важно: `version.json` загружать последним, чтобы приложение не предложило
обновление до того, как новые установочные файлы появились на сайте.

## Обязательное обновление

В `version.json` изменить:

```json
"mandatory": true
```

Тогда окно обновления нельзя будет закрыть кнопкой «Позже».
