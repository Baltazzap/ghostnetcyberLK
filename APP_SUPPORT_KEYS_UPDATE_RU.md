# GhostNet App — обновление админки и поддержки

Добавлено:

- Полноценное окно чата поддержки для пользователя.
- Отдельная панель поддержки для саппортов.
- Саппорт/админ может открыть обращение как чат, ответить и закрыть его.
- Админ может полностью удалять пользователей.
- Админ может открыть ключи пользователя, выдать/продлить/удалить ключ.
- В промокоды добавлена кнопка «Очистить».
- Удаление промокодов теперь вызывает полное удаление через API.
