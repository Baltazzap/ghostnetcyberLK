# GhostNet App — фикс встроенного VPN

Исправлено:

- Статус `DISCONNECTED` больше не считается включённым VPN.
- При первом включении Android должен запросить системное разрешение VPN.
- Список серверов теперь берётся из нового API `/api/vpn/servers`.
- Имена серверов отображаются из 3x-ui панели: Лондон, Амстердам, Хельсинки и т.д.
- Старый способ через `subscription_url` оставлен как резервный fallback.

Важно: сначала обновите API архивом `GhostNet_API_Server_Builtin_VPN_Fix.zip`, потом собирайте приложение.
