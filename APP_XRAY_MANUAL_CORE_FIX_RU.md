# GhostNet Cyber VPN — встроенный VPN, ручной Xray-конфиг

Что изменено:

- Убран проблемный парсинг VLESS через стандартный `parseFromURL` для VLESS.
- Для `vless://` приложение теперь вручную собирает полноценный Xray JSON-конфиг.
- Поддержаны параметры:
  - `security=reality`
  - `flow=xtls-rprx-vision`
  - `pbk`
  - `sid`
  - `sni`
  - `fp`
  - `spx`
  - `type=tcp`
  - `type=ws`
  - `security=tls`
  - `path`
  - `host`
- Для `vmess/trojan/ss` оставлен старый резервный парсер.

Важно:

Если ключ работает в Hiddify / Happ / v2RayTun, но не работает во встроенном приложении, проблема была в обработке VLESS-ссылки внутри приложения. Эта версия передаёт ядру не криво распарсенную ссылку, а готовый Xray-конфиг.

После обновления:

1. Удалите старую версию приложения с телефона.
2. Установите новый APK.
3. Войдите в аккаунт.
4. Откройте вкладку VPN.
5. Нажмите «Обновить».
6. Выберите сервер.
7. Нажмите «Включить VPN».

Если после этого клиент всё равно не появляется «В сети», значит текущий Flutter-плагин не поддерживает нужный Xray/REALITY режим на уровне встроенного ядра. Тогда следующий шаг — отдельная native-сборка на libbox/sing-box без flutter_v2ray.
