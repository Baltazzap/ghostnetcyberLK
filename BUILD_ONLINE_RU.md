# Как собрать приложение без установки Flutter на свой ПК

В этом проекте уже добавлена автосборка через GitHub Actions.
Тебе не нужно ставить Flutter, Android Studio или Visual Studio на компьютер.

## Что получится

После запуска сборки GitHub сам создаст:

- Android APK: `app-release.apk`
- Windows-версию: архив с `.exe` и нужными `.dll`

## Что сделать

1. Зарегистрируйся или войди на GitHub.
2. Создай новый репозиторий, например `ghostnet-cyber-vpn-app`.
3. Загрузи все файлы из этой папки в репозиторий.
4. Открой вкладку `Actions`.
5. Выбери сборку:
   - `Build Android APK`
   - `Build Windows EXE`
6. Нажми `Run workflow`.
7. Дождись завершения сборки.
8. Открой завершённый запуск и скачай файл в разделе `Artifacts`.

## Где скачать готовые файлы

После сборки внизу страницы запуска появится блок `Artifacts`.
Там будут:

- `GhostNet-Cyber-VPN-APK`
- `GhostNet-Cyber-VPN-Windows`

## Важно

Windows-версию нужно раздавать целой папкой/архивом, а не только `.exe`, потому что рядом нужны `.dll` файлы.

APK можно отправлять пользователям напрямую, но Android может показать предупреждение, что приложение установлено не из Google Play. Это нормально для APK, установленного вручную.
