# GhostNet Cyber VPN — обновление входа и адаптации экранов

Что изменено:

- На телефоне приложение теперь сразу показывает компактное окно входа / регистрации.
- Большой промо-блок убран с мобильного экрана входа, чтобы не приходилось прокручивать вниз.
- Добавлены вкладки «Вход» и «Регистрация».
- Экран входа адаптирован под маленькие телефоны и низкие экраны.
- Улучшена адаптация заголовков, логотипа и карточек под разные размеры экранов.
- Добавлен корректный `resizeToAvoidBottomInset`, чтобы клавиатура не перекрывала поля.
- Добавлен `keyboardDismissBehavior`, чтобы клавиатура скрывалась при прокрутке.

Как обновить проект:

1. Распакуйте архив.
2. Скопируйте файлы поверх текущего проекта.
3. В GitHub Desktop нажмите `Commit to main`.
4. Нажмите `Push origin`.
5. Запустите сборку APK / Windows EXE в GitHub Actions.
