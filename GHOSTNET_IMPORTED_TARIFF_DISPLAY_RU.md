# GhostNet 1.0.11+12 — тариф из вставленной подписки

- Ссылка `https://sub.ghostnetcyber.ru/...` сопоставляется с подписками пользователя из API.
- Актуальное название тарифа, срок и лимит устройств сохраняются локально.
- На главной Android/Windows показывается тариф именно вставленной подписки.
- В кабинете Android/Windows карточка подписки также показывает этот тариф.
- При обновлении или удалении ссылки данные автоматически обновляются.
- Если API временно недоступен, используются проверенные метаданные заголовков подписки и сохранённый локальный кэш.
