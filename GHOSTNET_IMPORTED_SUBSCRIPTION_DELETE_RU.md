# Удаление вставленной подписки — Android / Windows 1.0.12+13

В разделе «Кабинет → Мои ключи» добавлено полноценное удаление вручную вставленной подписки GhostNet.

При удалении:
- показывается окно подтверждения;
- удаляется ссылка `sub.ghostnetcyber.ru` с устройства;
- очищаются определённый тариф, срок, лимит устройств и кэш метаданных;
- главная страница и кабинет обновляются сразу;
- приложение возвращается к отображению обычной подписки аккаунта, если она есть.

Важно: удаление является локальным. Реальная подписка пользователя в GhostNet API не отменяется и не удаляется.
