# Исправление проверки Inno Setup в GitHub Actions

## Причина ошибки

Команда:

```powershell
& $isccPath /?
```

показывает справку Inno Setup, но `ISCC.exe` завершает её с кодом `1`.
GitHub Actions считает любой ненулевой код ошибкой и останавливает workflow,
хотя сам компилятор установлен и работает.

## Исправление

Workflow больше не запускает `ISCC.exe /?`.

Теперь он:

1. находит `ISCC.exe`;
2. проверяет, что файл действительно существует;
3. выводит путь;
4. читает версию из свойств EXE;
5. переходит к реальной сборке установщика.

Фактическая работоспособность компилятора проверяется следующим шагом:

```text
Build Windows installer
```

где запускается `tools/build_windows_installer.py`.
