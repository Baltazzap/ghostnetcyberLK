# Обновление приложения GhostNet

Что изменилось:

- «Пробный доступ» выдаёт 24 часа через API.
- «Купить подписку» открывает меню тарифов.
- «Мои ключи» открывает кабинет.
- Промокоды: вместо «Скопировать» теперь «Применить».
- В подсказках оставлен только WELCOME.
- Время подписки отображается по МСК GMT+3.
- Поддержка открывается внутри приложения с выбором: приложение или Telegram.
- Telegram поддержки: https://t.me/baltazzap
- Для Baltazzap / baltazzap@gmail.com появляется админ-панель.
- Админ-панель: пользователи, роли, промокоды, обращения поддержки.
- После возврата с оплаты приложение показывает «Успешно оплачено» или «Оплата отменена».
